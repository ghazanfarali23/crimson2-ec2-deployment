🚀 Extra Credit 2 — End-to-End Cloud Product Deployment

This panel documents how the capstone dashboard is delivered as a cloud product: static hosting on AWS Amplify, user authentication with Amazon Cognito, and the supporting AWS edge and storage services that keep the app fast and secure.

☁️ (1) AWS services in the stack
  • Amazon Cognito User Pool — sign-up, sign-in, password recovery; issues JWT access / ID tokens for the browser.
  • AWS Amplify Hosting — CI/CD from Git, HTTPS, global distribution via CloudFront with S3 origin behind the scenes.
  • Amazon S3 — stores static objects (HTML, CSS, JS) for each deployment; not accessed directly by end users when served through Amplify.
  • Amazon CloudFront — CDN caching and TLS termination at the edge (provided as part of Amplify Hosting).
  • AWS Amplify JS (CDN) — browser client configured with only the public User Pool ID and app client ID (no client secret in SPA flows).
💵 (2) Estimated monthly cost
ItemFree tier / notesRough scale
Cognito MAUsFirst 50k MAUs / month free (see current pricing)Often $0 for class demos
Amplify HostingBuild minutes + hosting GB-mo; small static sites often remain low~$0–5/mo light traffic
S3 + CloudFrontBundled into Amplify hosting charges; pay for egress beyond free tiersLow until traffic spikes

Figures are order-of-magnitude only; use the AWS Pricing Calculator and your own usage reports for commitments.

🔐 (3) Security assumptions
  • HTTPS only in production (Amplify default); protects tokens and credentials in transit.
  • Cognito validates passwords and MFA if enabled; the SPA stores session state via Amplify, not custom cookies with secrets.
  • JWTs are validated by AWS for API calls; this dashboard is static-only, so tokens are mainly for gating access and future API integration.
  • No secrets in Git — only public IDs in index.html; never commit OAuth client secrets or IAM keys.
🛑 (4) Shutdown & budget limiting
  • AWS Budgets + billing alarms — email/SNS when forecasted or actual spend crosses a threshold (e.g. $5).
  • Disable Amplify app — stops builds and hosting charges for the frontend.
  • Delete Cognito user pool — removes auth and MAU metering after the course demo.
  • Review Cost Explorer and Free Tier usage monthly during the project.
